Hướng dẫn Setup VS Code 2022

Các bạn nhớ xem kĩ hướng dẫn và hình ảnh setup nhá

Bước 1: Các bạn truy cập theo link: https://code.visualstudio.com và chọn phiên bản phù hợp với máy tính của bạn và tiến hành tải về

Bước 2: Web sẽ tự động tải về file setup có đuôi '.exe'

Bước 3: Sau khi tải về, tiến hành chạy file cài đặt. Chỉ cần bấm next - next để setup nha

Bước 4: Sau khi setup xong các bạn mở VS Code sẽ có giao diện như sau

Bước 5: Các bạn bấm vô mục này có hình giống ô vuông chồng chồng lên nhau!

Bước 6: Tìm các extension như hình bên dưới và bấm install nhé
